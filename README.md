# hnews

> [Hacker News](http://news.ycombinator.com) CLI 

## Install

    $ gem install hnews


## Usage

[Demo GIF](http://f.cl.ly/items/3J2E2N2c0a2O3y3v1c1q/example.gif)

    # Display item lists
    $ hnews home
    $ hnews new
    $ hnews best

    # Open an item from last list in browser
    $ hnews open [item number]
